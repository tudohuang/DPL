About the Project

This project integrates millimeter-wave radar technology, modern electronics, and Python-based software to create a hands-on, immersive educational experience for students learning STEM subjects.

Features

High-Precision Sensing

Utilizes 24 GHz radar for real-time, non-contact motion detection.

Real-Time Analysis

Process data instantly with our Python scripts, delivering meaningful insights.

Modular Design

Flexible PCB layout and firmware for various experiments and configurations.

Explore the Hardware

Interact with our 3D PCB model to understand the hardware in detail.

Educational Value

This system supports STEM education by enabling students to learn about physics (SHM, resonance), electronics (circuit design), and programming (data analysis).
